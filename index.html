Melotrack is a musical quiz. Recognize the music, solve the riddle and type the answer. Play starts and stops playback, mode switches selection modes: repeat, next, previous. Bar displays progress and rewinds playback, volume displays and changes level. Arrows switch tours, name is displayed in the index. Time displayed in the timer, when it ends, tour checks. Use clef to check ahead. Caption contains various information: instruction, description or rules. Together with the header they make up a riddle. Keys to the riddles are the column names of the table, tracks refer to table rows. Intersections have inputs for answers. Have fun and good luck :)
